### TaskFlow - Projeto Integrador (2ª Etapa)

### 📌 Descrição

O **TaskFlow** é um sistema web desenvolvido como Prova de Conceito (PoC) para o Projeto Integrador do SENAC.
O objetivo é permitir que usuários cadastrem e organizem tarefas, definam prazos e compartilhem atividades com outros usuários.

### 🚀 Tecnologias Utilizadas

* Frontend: HTML5, CSS3, JavaScript, Bootstrap
* Backend: PHP 8
* Banco de Dados: MySQL
* Servidor Local: XAMPP
* Versionamento: Git/GitHub

### 📂 Estrutura do Projeto

* `frontend/` → telas do sistema
* `backend/` → API PHP e controle de sessão
* `database/` → script SQL para criação do banco

### 🛠 Como executar o projeto (XAMPP)

### 1\. Instalar e iniciar o XAMPP

* Inicie os serviços **Apache** e **MySQL**

### 2\. Criar banco de dados

1. Acesse `(http://localhost:8090/phpmyadmin)`
2. Importe o arquivo:

   * `database/script.sql`

### 3\. Colocar o projeto no htdocs

Copie a pasta do projeto para:

`C:\xampp\htdocs\ProjetoIntegrador_TaskFlow`

### 4\. Abrir no navegador

Acesse:

`(http://localhost:8090/ProjetoIntegrador_TaskFlow/frontend/index.html)`

### 👤 Funcionalidades da Prova de Conceito

* Cadastro de usuário
* Login com sessão
* Criar tarefas
* Listar tarefas
* Editar tarefas
* Excluir tarefas
* Compartilhar tarefas com outro usuário
